# 実装済み機能

日本語 | [English](features-en.md)

## エディター機能

- アイコンパレットからの配置 (`public/icons` を自動検出)
- サンプルアイコン (`public/icons-sample`) の表示切り替え
- ボックス / テキスト / 矢印 / ラインの追加
- 選択、移動、サイズ変更
- Z-Order の前面 / 背面移動
- 複製・削除
- 編集履歴の保持 (設定で保持件数を調整)

## 保存と共有

- ローカルストレージへの JSON 保存
- JSON のエクスポート / インポート
- PNG 形式でのエクスポート
- Cosmos DB 設定時は `/api/diagrams` でクラウド保存
- 履歴スナップショットをローカル保存し `/history` で復元

## 画面とナビゲーション

- `/` : ランディングページ
- `/editor` : 図の編集画面
- `/items` : アイコン一覧と検索 ( `/api/icons` )
- `/history` : ローカル履歴の一覧と復元
- `/settings` : 履歴保持件数と PNG エクスポート倍率の設定
- `/about` : アプリ概要

## 共通機能

- 言語切り替え (日本語 / 英語)
- `/api/health` によるヘルスチェック
